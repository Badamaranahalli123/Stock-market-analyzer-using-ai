# Stock Market Predictor AI

### Description
A Python-based tool to analyze stock market trends and predict future prices using historical data.

### Tech Stack
* **Language:** Python 3
* **Libraries:** Pandas, Matplotlib, Scikit-Learn
* **Model:** Linear Regression

### Features
* Fetches historical stock data.
* cleans missing values automatically.
* Visualizes price trends.
* Predicts next day's closing price.
